CSS - Vantagens, Benefícios e Funcionalidades

Vantagens

Com CSS economiza-se tempo de criação e manutenção (isola os códigos de formatação aplicado a várias páginas HTML em um único arquivo ".css");
Reduz código de descritores HTML da página (tags);
Os navegadores (browsers) carregam mais rápido;
Maior eficiência no gerenciamento do layout;
CSS é simples pois descreve apenas estilos;
Com recursos mais avaçados de CSS pode-se conseguir um design sofisticado sem utilização de imagens e tabelas;
Quase tudo o que se pode fazer com CSS pode ser feito também com os descritores de HTML convencionais (com poucas exceções).

Benefícios

CSS é uma revolução no mundo da web design. Os benefícios concretos do CSS incluem:
Controle do layout de vários documentos de uma única folha de estilo;
Um controle mais preciso do layout;
Aplicação de diferentes layouts para diferentes mídias (tela, impressora, etc);
Numerosas técnicas avançadas e sofisticadas.
CSS é suportado por todos os navegadores de hoje

Funcionalidades

O css é uma linguagem de estilos que faz a definição dos documentos HTML.
Com ele, é feito o controle das fontes, cores, margens, linhas(…)de uma página WEB. O HTML é encarregado de dar a estrutura ao conteúdo, o CSS faz a personalização desse mesma estrutura.Ou seja, o CSS dá o estilo para a página HTML.
